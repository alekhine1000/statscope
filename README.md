StatScope: A Descriptive Statistics Diagnostic Tool
What it Does
StatScope is a simple, browser-based tool developed to help educators teach descriptive statistics in a more intuitive and interactive way. It allows users to upload an Excel  file, select a numeric column, and instantly see what the data is really saying—no coding, software installation, or setup required.
The idea is straightforward: remove the technical barriers so that students can focus on understanding the concepts.
Statistical Summary
StatScope computes the key descriptive statistics needed to make sense of any dataset:
•	Central Tendency
•	Dispersion Measures.
•	Distribution Shape
•	Outlier Detection

Visualizations
•	Histogram
A clear visual of how data is distributed, with an overlaid normal curve for easy comparison. Students can quickly identify skewness, clustering, or multiple peaks.
•	Box-and-Whisker Plot
A compact summary of the data using the five-number summary (minimum, Q1, median, Q3, maximum). Outliers are shown explicitly, making them easy to spot and discuss.
How Users Benefit
StatScope was designed with teaching and learning in mind. For educators, it serves as a live demonstration tool—something you can use in class to make abstract statistical ideas more concrete. For students, it provides a way to explore data independently without needing to learn Python, R, or complex spreadsheet functions.
What would normally take several steps in traditional software happens instantly. This allows more time for interpretation, discussion, and critical thinking.
Because it runs entirely in a web browser, it works across devices—Windows, Mac, Linux, or Chromebook—making it ideal for flexible and online learning environments.
In Short
Open the browser. Upload the data. Diagnose the dataset.
That’s the workflow—and that’s the point.
